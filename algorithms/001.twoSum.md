## Solution 
Python3
```python3
class Solution:
    def twoSum(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: List[int]
        """
        checked = {}
        for i in range(len(nums)):
            if nums[i] in checked:
                return [checked[nums[i]], i]
            else:
                checked[target - nums[i]] = i
```
## Complexity: O(n)
